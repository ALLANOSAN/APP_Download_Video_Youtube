# Política de Privacidade - YouTube Search Pro

**Última atualização:** Janeiro de 2026

## Resumo

A extensão YouTube Search Pro **não coleta, armazena ou compartilha** nenhum dado pessoal dos usuários.

## Dados Coletados

### O que NÃO coletamos:
- ❌ Informações pessoais
- ❌ Histórico de navegação
- ❌ Dados de identificação
- ❌ Cookies de rastreamento
- ❌ Informações do dispositivo

### Armazenamento Local

A extensão usa o `chrome.storage.local` **apenas** para:
- Salvar a última busca realizada (para conveniência do usuário)
- Os dados ficam **apenas no seu computador**
- Nenhum dado é enviado para servidores externos

## Permissões Utilizadas

### `storage`
- **Motivo:** Salvar preferências localmente no navegador do usuário
- **Dados:** Apenas o texto da última busca

### `host_permissions` (youtube.com)
- **Motivo:** Realizar buscas de vídeos no YouTube
- **Uso:** A extensão faz requisições para youtube.com para buscar resultados de pesquisa
- **Dados:** Apenas a consulta de busca é enviada ao YouTube (como qualquer busca normal)

## Compartilhamento de Dados

**Não compartilhamos nenhum dado com terceiros.**

A extensão apenas:
1. Envia sua busca para o YouTube (como se você buscasse manualmente)
2. Exibe os resultados na popup

## Seus Direitos

Como não coletamos dados, não há dados pessoais para:
- Acessar
- Corrigir
- Excluir

Para limpar os dados locais (última busca), você pode:
1. Remover a extensão
2. Limpar dados do navegador

## Alterações nesta Política

Quaisquer alterações serão publicadas nesta página com a data atualizada.
