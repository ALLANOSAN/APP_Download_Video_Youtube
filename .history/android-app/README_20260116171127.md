# YouTube Downloader Pro - Android App

App Android para baixar vídeos e músicas do YouTube, desenvolvido com **Kivy** (Python).

## 📱 Funcionalidades

- 🔍 **Pesquisa no YouTube** - Digite e busque vídeos
- 📋 **Colar URL** - Cole links do YouTube diretamente
- 🎵 **Download de Áudio** - Formatos: MP3, M4A, OPUS, WAV, FLAC
- 🎬 **Download de Vídeo** - Qualidades: 360p até 4K
- 📊 **Bitrate selecionável** - 96k até 320k
- 🎨 **Tema escuro** - Interface moderna e bonita

## 🛠 Como Compilar

### Pré-requisitos

```bash
# Instale o uv (gerenciador de pacotes Python moderno)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Instale o Buildozer
uv tool install buildozer

# Dependências do sistema (Ubuntu/Debian)
sudo apt install -y \
    build-essential \
    git \
    ffmpeg \
    libsdl2-dev \
    libsdl2-image-dev \
    libsdl2-mixer-dev \
    libsdl2-ttf-dev \
    libportmidi-dev \
    libswscale-dev \
    libavformat-dev \
    libavcodec-dev \
    zlib1g-dev \
    libgstreamer1.0-dev \
    gstreamer1.0-plugins-base \
    gstreamer1.0-plugins-good \
    openjdk-17-jdk \
    autoconf \
    automake \
    libtool \
    libffi-dev \
    cmake \
    unzip

# Arch Linux / CachyOS
sudo pacman -S --needed \
    base-devel \
    git \
    ffmpeg \
    sdl2 \
    sdl2_image \
    sdl2_mixer \
    sdl2_ttf \
    portmidi \
    gstreamer \
    gst-plugins-base \
    gst-plugins-good \
    jdk17-openjdk \
    cmake \
    unzip
```

### Compilar APK (Debug)

```bash
cd android-app

# Primeira vez (demora ~30 minutos, baixa Android SDK/NDK)
buildozer android debug

# APK estará em: bin/youtubedownloaderpro-1.0.0-arm64-v8a-debug.apk
```

### Compilar APK (Release)

```bash
# Gera APK otimizado e assinado
buildozer android release

# Para assinar manualmente:
# jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore my-release-key.keystore bin/*.apk alias_name
```

### Instalar no Celular

```bash
# Via ADB (com celular conectado em modo debug)
buildozer android deploy run

# Ou transfira o APK manualmente
adb install bin/youtubedownloaderpro-1.0.0-arm64-v8a-debug.apk
```


## ⚠️ Observações Importantes

1. **FFmpeg**: O app precisa do FFmpeg para converter áudio. No Android, é embutido automaticamente pelo yt-dlp.

2. **Permissões**: O app pede permissão de armazenamento para salvar os downloads.

3. **Diretório de Download**:
   - Android: `/storage/emulated/0/Download/YouTubeDownloader/`
   - Desktop: `~/Downloads/YouTubeDownloader/`

4. **Tamanho do APK**: ~50-80MB devido às dependências (Python, Kivy, yt-dlp).

5. **Tempo de compilação**: A primeira compilação demora ~30 minutos pois baixa Android SDK, NDK e compila Python.

## 🐛 Problemas Comuns

### "Recipe does not exist"

```bash
# Limpe o build e tente novamente
buildozer android clean
buildozer android debug
```

### "SDK/NDK not found"

```bash
# O buildozer baixa automaticamente, mas se falhar:
export ANDROIDSDK=~/.buildozer/android/platform/android-sdk
export ANDROIDNDK=~/.buildozer/android/platform/android-ndk-r25b
```

### "yt-dlp não encontrado"

Verifique se `yt-dlp` está nos requirements do buildozer.spec.
